En la estructura del repositorio estan las 10 carpetas con sus respectivos archivos .gitKeep.
cada carpeta representa una instancia del proyecto final.
elegi esta organizacion por que me parecio mas ordenada
para el nivel tres utilize:
1. <b></b> para poner texto en negrita
2. <u></u> para subrayar el titulo
3. <th></th> para agregarle un encabezado a la tabla
4. <input required> para que el campo de texto sea obligatorio
5. <s></s> para tachar texto
6. <h2></h2> para un encabezado mas chico
